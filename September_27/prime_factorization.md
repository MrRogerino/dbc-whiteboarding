## Prime Factorization

Given a positive number n > 1 find the prime factor decomposition of n. The result will be a string with the following form :

```ruby
 "(p1**n1)(p2**n2)...(pk**nk)"
```

### Examples

```ruby
prime_factorization(86240)
=> ""(2**5)(5)(7**2)(11)""

prime_factorization(200)
=> ""(2**3)(5**2)""
```
